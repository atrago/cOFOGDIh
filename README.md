# 项目简介

## 核心功能
- 用户管理：三角色体系（农户/农技员/管理员），JWT登录认证，权限控制
- AI病虫害识别：对接豆包大模型API，上传图片智能诊断，历史记录管理
- 价格行情：实时价格发布，ECharts趋势图表，多品类切换
- 种植日历：农事安排管理，按月查看，支持增删改查
- 教学视频：视频分类管理，在线播放，农技员可发布
- 社区互动：帖子发布/评论，置顶功能，图片上传
- 案例库：种植案例分类浏览，农技员贡献，管理员审核
- 行业资讯：农业新闻发布与展示
- 私信系统：用户间实时私信，会话列表，未读提醒

## 适用场景
- 农户：获取AI病虫害识别、价格行情、种植日历等信息，提高农业生产效率。
- 农技员：使用AI病虫害识别功能诊断作物疾病，管理农事活动，分享教学视频。
- 管理员：管理用户，审核案例，发布行业资讯，维护平台运营。

## 技术特点
- 前端：Vue 3.3 + Vite 4 + Element Plus + ECharts 5
- 后端：Spring Boot 2.7 + MyBatis + JWT认证
- 数据库：MySQL 8.0
- 架构：前后端分离 + RESTful API + SSM三层架构
